# Genome GC Landscape Analysis

## Biological question:
How does GC content vary across bacterial genomes?

## Organisms:
1. Escherichia coli
2. Helicobacter pylori

## Data:
Genomes were downloaded from NCBI.

## Methods:
1 Bash was used to compute genome size.
2 R  was used for sliding window GC analysis and plotting.

## Results og GC content present in each organism:
E coli GC ~50%
H pylori GC ~39%

## Interpretation:
Different GC composition reflects evolutionary differences in bacterial genomes.
