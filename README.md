# Genome GC Landscape Analysis

## Biological question
How does GC content vary across bacterial genomes?

## Organisms
Escherichia coli
Helicobacter pylori

## Data
Genomes downloaded from NCBI.

## Methods
1 Bash used to compute genome size
2 R used for sliding window GC analysis

## Results
E coli GC ~50%
H pylori GC ~39%

## Interpretation
Different GC composition reflects evolutionary differences in bacterial genomes.
